# Vazarbot


## Commands:


* **Notification:**

![](https://github.com/dht2003/vazarbot/blob/main/media/notification.gif)

```
!notification in 10m Standup
!notification at 14:30 Important message
```

* **doctor/paramedic**

![](https://github.com/dht2003/vazarbot/blob/main/media/doctor.gif)

```
!paramedic 67 in 3s
!paramedic 67 in 3s
!doctor 67 at 15:15
!paramedic 67 at 15:15
```

* **popup:**
![](https://github.com/dht2003/vazarbot/blob/main/media/popup.gif)
```
!popup "important msg"
```

* **config**

```
!config show
!prefix <new prefix>
```

* **Help:**
```
!help
!help <command>
```
